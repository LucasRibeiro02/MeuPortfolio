A solução foi construída utilizando Power Automate, Microsoft Approvals e Microsoft Teams, permitindo que o usuário envie o orçamento em PDF diretamente por uma interface simples, informando apenas o gestor responsável. A partir disso, o fluxo automatizado: - Encaminha a solicitação automaticamente para aprovação - Armazena o documento de forma segura na nuvem (OneDrive) - Registra todo o histórico de aprovação - Atualiza o status em tempo real - Permite aprovação com um clique
